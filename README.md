Project Overview

This is a Django-based portfolio project that showcases my web development skills.
It includes multiple projects I have built, along with my skills, experiences, and contact information.

Features

Built with Django framework.

User authentication (Login/Signup).

Display projects dynamically.

Contact form integrated with email functionality.

Responsive design for mobile and desktop.

Clean UI to showcase portfolio projects.

Technologies Used

Backend: Django, Python

Frontend: HTML, CSS, Bootstrap (or Tailwind)

Database: SQLite (default, can switch to PostgreSQL)

Other Tools: Git, GitHub

Installation

Clone the repository:

git clone https://github.com/Magferati/My-Port-by-django.git


Navigate to the project folder:

cd django-portfolio


Create a virtual environment:

python -m venv env


Activate the virtual environment:

Windows: env\Scripts\activate

Mac/Linux: source env/bin/activate

Install dependencies:

pip install -r requirements.txt


Apply migrations:

python manage.py migrate


Run the development server:

python manage.py runserver


Open in browser: http://127.0.0.1:8000/
